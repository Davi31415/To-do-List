# To-Do List
Este projeto é um gerenciador de tarefas criado para facilitar a organização da rotina diária. A aplicação permite cadastrar, acompanhar e concluir tarefas de forma prática e rápida, buscando produtividade através de uma interface minimalista.

-Nota: Como o projeto utiliza serviços de hospedagem gratuitos para o deploy, o primeiro carregamento ou algumas requisições podem demorar alguns segundos a mais para responder.

**Acesso o projeto:** https://to-do-list-ten-sigma-52.vercel.app/

## ✅ Funcionalidades

O projeto conta com os seguintes recursos principais:

* **Criação de Tarefas:** Adição de novas pendências à lista.
* **Controle de Status:** Opção de marcar tarefas como concluídas ou reabrí-las.
* **Exclusão de Itens:** Remoção de tarefas antigas ou desnecessárias da listagem.
* **Persistência de Dados:** Salvamento das tarefas para que não se percam ao atualizar a página.
* **Autenticação de Usuários:** Sistema de cadastro e login para garantir que cada usuário tenha acesso apenas às suas próprias tarefas.
